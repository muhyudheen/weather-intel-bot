# Weather Intel Bot

An AI-powered weather chatbot that detects weather-related queries, fetches real-time data from Open-Meteo, and synthesizes it into natural conversational responses.

## Features
- Current weather conditions (temperature, humidity, wind, UV index)
- 7-day forecast
- Works for any city in the world
- Handles ambiguous city names

## Built With
- FastAPI
- Open-Meteo API (no API key needed)
- AI Pipe (OpenAI-compatible LLM)
